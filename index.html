Read Aloud

Listen to web pages with word highlighting

Overview

Read Aloud is a Chrome extension that converts text on web pages to speech, making content more accessible and convenient to consume. Whether you're multitasking, have visual impairments, or simply prefer listening to reading, Read Aloud enhances your browsing experience.

Features

Word Highlighting

Follow along as each word is highlighted in real-time while being read aloud, making it easy to track progress.

Floating Control Bar

Convenient controls for play/pause, stop, and settings that stay visible as you navigate the page.

Read from Selection

Right-click on selected text to start reading from that specific point on the page.

Customizable Voice

Choose from available system voices and adjust speech rate and pitch to your preference.

Smart Text Extraction

Intelligently identifies and reads only the visible, meaningful content on web pages.

Easy to Use

Simple interface with intuitive controls that work across all websites.

Installation

Read Aloud is available as a Chrome extension. Follow these simple steps to install:

  1. Download the extension files from the GitHub repository
  2. Open Chrome and navigate to chrome://extensions
  3. Enable "Developer mode" by toggling the switch in the top-right corner
  4. Click "Load unpacked" and select the extension directory
  5. The Read Aloud extension icon should now appear in your browser toolbar

Once installed, simply click the extension icon or right-click on selected text to start using Read Aloud.

How to Use

Using Read Aloud is simple:

  • Start Reading: Click the extension icon and then "Start Reading" to begin from the top of the page
  • Read Selection: Select text on the page, right-click, and choose "Read from this text"
  • Control Playback: Use the floating control bar to play/pause, stop, or access settings
  • Adjust Settings: Click the settings icon in the control bar to change voice, speed, and pitch
