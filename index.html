Read Aloud

A Chrome extension that reads web pages aloud with word highlighting

About

Read Aloud is a powerful Chrome extension that converts text on any webpage into speech, making content more accessible and convenient to consume. Whether you're multitasking, have visual impairments, or simply prefer listening to reading, Read Aloud enhances your browsing experience.

Key Features

  • Text-to-Speech: Converts any webpage text into natural-sounding speech
  • Word Highlighting: Highlights each word as it's being read for easy following
  • Convenient Controls: Floating control bar with play, pause, and settings
  • Read Selection: Right-click on selected text to start reading from that point
  • Customizable: Adjust reading speed, pitch, and choose from available voices
  • Simple Interface: Easy-to-use popup with one-click reading start

Installation Instructions

  1. Download the extension from the Chrome Web Store (link coming soon)
  2. Click "Add to Chrome" to install the extension
  3. Once installed, you'll see the Read Aloud icon in your browser toolbar
  4. Navigate to any webpage and click the icon to start reading
  5. Alternatively, select text on a page, right-click, and choose "Read from this text"

How to Use

Using Read Aloud is simple:

  • Click the Read Aloud icon in your toolbar to start reading from the top of the page
  • Select specific text and right-click to use the "Read from this text" context menu option
  • Use the floating control bar to pause/resume reading or access settings
  • Adjust reading speed, pitch, and voice in the settings panel
